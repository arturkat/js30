*** Quick start ***
    bower install
    npm install
    gulp
    'Livereload' in chrome


** start Bower **
    bower install
    bower install <package> --save

** start NPM **
    npm install
    npm install --save <module>


____Run It____
    gulp
    enable 'Livereload' in chrome


*******************
** From Scratch **
    ** Bower **
    npm install -g bower
    bower init    //to create initial file
    bower install jquery
    bower install <package> --save
    bower install <package> --save-dev

    ** NPM **
    npm init
    npm install
    npm install 

    ** Gulp **
    npm install --global gulp
    npm install --save-dev gulp
    npm install --save-dev <gulp-concat-css>    //install package
    ** Run __
    gulp <task> <othertask>
